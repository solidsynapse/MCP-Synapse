# Contributing to MCP Synapse

## Reporting Bugs
Use GitHub Issues and fill out the bug report template.

## Requesting Features
Use GitHub Issues and select the "feature request" template.

## Branding
The MCP Synapse name and logo belong to Solid Synapse.
Forks and derivative projects must not imply official distribution.

## Source Visibility
This project is not open source. Source code is publicly viewable, but the
external contribution flow is not active yet.
