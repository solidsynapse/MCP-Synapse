# Trademark Baseline Policy

Status: Active
Owner: Product/Legal

## Official identity
- Product name and logo identify official MCP Synapse distribution channels.

## Fork/re-distribution baseline
- Forks may not use product name/logo in a way that implies official distribution or endorsement.
- Modified builds must be clearly labeled as unofficial.
- Official branding assets are reserved for official releases.

## Public communication baseline
- Official website/docs/release channel must be explicitly identified.
- Security reports should be directed to `security@solidsynapse.com`.

## Scope note
This baseline is a practical release policy. Formal legal expansion can be added later without changing current release controls.
